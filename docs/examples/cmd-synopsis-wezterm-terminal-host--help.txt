Manage the Windows default terminal host configuration. Errors out on non-Windows platforms

Usage: wezterm terminal-host <COMMAND>

Commands:
  enable   Enable WezTerm as the Windows default terminal
  disable  Disable WezTerm as the Windows default terminal and restore the previous selection
  help     Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
